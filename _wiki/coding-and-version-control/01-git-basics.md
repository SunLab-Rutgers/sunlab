Use this page for the group’s recommended Git workflow.

### Basic Workflow

```bash
git status
git pull
git checkout -b feature/my-change
git add .
git commit -m "Describe the change"
git push
```

Prefer small commits with clear messages. Add project-specific branch and pull request rules here.
